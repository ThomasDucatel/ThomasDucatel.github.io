# PROFIL DE CHERCHEUR

Je travaille dans le domaine des **matériaux organiques pour l’opto‑électronique**, avec une expertise centrée sur :

- les **polymères semi‑conducteurs orientés**,  
- les **modulateurs spatiaux de lumière photovoltaïques**,  
- les **interactions ioniques dans les dispositifs électro‑optiques**,  
- les **films minces organiques** pour applications photoniques.

Mon approche combine **fabrication**, **caractérisation opto‑électronique**, **analyse ionique**, **mesures optiques**, et **modélisation** sous MATLAB/Python.  
Je m’intéresse particulièrement aux liens entre **structure**, **propriétés optiques** et **fonctionnement des dispositifs**.
